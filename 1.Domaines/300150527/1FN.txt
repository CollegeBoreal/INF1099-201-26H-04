1NF – Première Forme Normale

Domaine : Application e-commerce (Ultras Green Army Shop)

Dans la première forme normale, toutes les données sont atomiques,
mais certaines informations sont encore regroupées dans une seule table.

TABLE : ORDER_DETAILS

- orderId
- orderDate
- userName
- userPhone
- userAddress
- productName
- productPrice
- quantity
- paymentMethod

Cette table respecte la 1NF car :
- Chaque champ contient une seule valeur
- Il n’existe aucun champ multivalué

Cependant, elle contient beaucoup de redondance de données
(informations utilisateur et produit répétées).
